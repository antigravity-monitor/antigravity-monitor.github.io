Antigravity Monitor

Cloud Code quota dashboard

ANTIGRAVITY CONSOLE COMPANION

Track Claude and Gemini quotas in real time.

This is a static GitHub Pages app. It signs you in with Google via GSI, calls Cloud Code internal endpoints, and renders your per-model reset timers and remaining quotas.

Sign in with your Google account to view your Antigravity quota. No data is stored on any server.

WHAT YOU GET

  • Subscription tier badge (free / pro / ultra)
  • Per-model quota bars, remaining percentage
  • Reset countdowns and auto-refresh every 30s
